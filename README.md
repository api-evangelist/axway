# Axway

Axway provides API management, integration, and security solutions enabling organizations to connect, secure, and manage APIs across hybrid IT environments. Axway offers the Amplify platform for API management and a range of integration and security products for enterprises.

## APIs

### Axway Amplify Platform API
APIs for managing API products, environments, and consumer subscriptions across the enterprise API management platform.
- **Documentation**: https://docs.axway.com/

### Axway Amplify Central API
Unified API management hub for discovery, consumption, and governance of APIs across the enterprise.
- **Documentation**: https://docs.axway.com/bundle/amplify-central/

### Axway API Gateway
Enterprise-grade API gateway providing security, mediation, and traffic management for APIs.
- **Documentation**: https://docs.axway.com/bundle/axway-open-docs/

### Axway API Manager
Web-based management interface for registering, virtualizing, and publishing APIs to consumers.
- **Documentation**: https://docs.axway.com/bundle/axway-open-docs/page/docs/apim_administration/apimgr_admin/index.html

### Axway API Builder
Low-code tool for building and deploying microservices and APIs from data sources and business logic.
- **Documentation**: https://docs.axway.com/bundle/api-builder/

### Axway Amplify Streams
Real-time event streaming API platform with SSE and WebSocket connections.
- **Documentation**: https://docs.axway.com/bundle/amplify-streams/

### Axway Flow Manager
Managed file transfer and B2B integration orchestration solution.
- **Documentation**: https://docs.axway.com/bundle/FlowManager_20_allOS_en_HTML5/

## Features

- **API Lifecycle Management** — Manage the full API lifecycle from design to retirement.
- **API Gateway** — Enterprise-grade gateway with policy-based security and OAuth.
- **Unified API Catalog** — Centralized catalog for discovering and consuming APIs from any gateway.
- **Event Streaming** — Real-time event streaming with SSE and WebSocket.
- **Managed File Transfer** — Automate B2B file exchange with SLA governance.
- **Low-Code API Building** — Build microservices visually with API Builder.
- **Consumer Self-Service** — Enable API consumers to discover and subscribe independently.
- **Hybrid Deployment** — Deploy across on-premises, cloud, and hybrid environments.

## Use Cases

- **Enterprise API Management** — Govern and manage APIs across distributed enterprise environments.
- **B2B Integration** — Automate partner data exchange with secure MFT and EDI workflows.
- **API Monetization** — Publish APIs to a marketplace with tiered subscriptions and billing.
- **Real-Time Data Streaming** — Deliver live data updates via event streaming APIs.
- **Open Banking** — Implement open banking APIs compliant with PSD2 and regulations.
